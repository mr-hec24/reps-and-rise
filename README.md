# Reps & Rise
Phoenix-Soteria's new product, a gym note taking app developed to track and provide data in clean visuals for your gains.

Here is the stack we will be using throughout this project:
- Expo (Frontend): https://docs.expo.dev/
- PostHog (Cloud)
- Supabase (Database)

## How to Run & Test This App
1. Open up the project on your device
2. Navigate into \reps-and-rise\reps-and\rise
3. In the command line, run $npx expo start$.
4. Pull out your phone and scan the QR code
 - This should take you to Download and Open the Expo Go App
 5. In the terminal, press 'w' to open the web version of the app too
